DMSA

Cálculo da Captação Absoluta do DMSA

Baixar .zip Baixar .tar.gz Veja-o no GitHub

Esse software, livre e gratuito, serve para o cálculo da captação absoluta do DMSA em medicina nuclear. É ainda um software em desenvolvimento e não testado, portanto, use-o por sua própria conta e risco.

Esse projeto de software foi desenvolvido em Qt e C++ com o intuito de aprendizado e treinamento dessas tecnologias. É distribuido gratuitamente seu código binário (executáveis) juntamente com seu respectivo código fonte. É muitiplataforma e pode ser compilado para MacOSX, Windows, e Linux. Aos que desejam usá-lo sem ter de compilá-lo, existe uma versão compilada para MacOSX e outra para Windows dentro da pasta "executable". O dmsa.app é a versão para MacOSX é o terminado com a extensão ".exe" é a versão para Windows. A versão para windows, dmsa-qt-win.exe aqui, pode ser baixada diretamente.

Aos que temem executar código fechado (dmsa.app ou dmsa-qt-win.exe), com razão, a opção é baixar o Qt Creator em http://qt-project.org e importar o projeto DMSA para dentro do Qt e executá-lo (Build/Run). Assim, pode-se executar em sua máquina, um código que vc pode inspecionar.

Sou muito grato aos que me reportarem erros e sugestões de melhorias.
