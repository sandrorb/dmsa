DMSA

Cálculo da Captação Absoluta do DMSA

Baixar .zip Baixar .tar.gz Veja-o no GitHub

Esse software serve para o cálculo da captação absoluto do DMSA em medicina nuclear. É ainda um software em desenvolvimento e não testado, portanto, use-o por sua própria conta e risco.

Esse projeto de software foi desenvolvido em Qt e C++ com o intuido de aprendizado e treinamento dessas tecnologias. É distribuido seus binários juntamente com seu respectivo código fonte gratuitamente. É muitiplataforma e pode ser compilado para MacOSX, Windows, e Linux. Os que desejam usá-lo sem ter que compilá-lo, existe uma versão compilada para MacOSX e outra para Windows dentro da pasta "executable". O dmsa.app é a versão para MacOSX é o terminado com a extensão ".exe" é a versão para Windows. Alternativamente, baixe o dmsa-qt-win-v0.4.2.exe aqui.

Aos que temem executar código fechado (dmsa.app ou dmsa*.exe), com razão, a opção é baixar o Qt Creator em http://qt-project.org e importar o projeto DMSA para dentro do Qt e executá-lo (Build/Run). Assim, pode-se executar em sua máquina, um código que vc pode inspecionar.

Ficaria muito grato aos que me reportarem erros e sugestões de melhorias.
